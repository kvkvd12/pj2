# 5i 스파르타 회식비 신청하기

## 프로젝트 주제
* 단체사진을 업로드하면 인원 수를 체크해 회식비를 얼마 받을수 있는지 조회하고 신청하는 사이트를 구현해본다.
1. 로그인 / 회원가입 / 로그아웃
2. 이미지 업로드 
3. 결과 화면 리스트 

(추가기능)
1. 회식비 신청 승인
2. 반려동물 간식비
3. 내 회식비 승인내역 모아보기
